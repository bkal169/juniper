The Juniper Rose publishing rhythm. Consistent output across platforms without burnout.

## Content Pillars

Each pillar has a recurring format that rotates weekly:

### Pillar 1: Real Estate
- **Week 1:** Market update / data breakdown (YouTube)
- **Week 2:** Agent tips / strategy (YouTube)
- **Week 3:** Client story / case study (Podcast)
- **Week 4:** Listing walkthrough or neighborhood spotlight (YouTube)

### Pillar 2: Wealth Building
- **Week 1:** Concept explainer (Blog + YouTube)
- **Week 2:** Tool/resource review (YouTube)
- **Week 3:** Personal finance story (Podcast)
- **Week 4:** Q&A / community questions (Live or YouTube)

### Pillar 3: Nonprofit / Heart of Juniper
- **Week 1:** Impact update / student spotlight (Social)
- **Week 2:** Grant or donor highlight (Email + Blog)
- **Week 3:** Mission story (Podcast segment)
- **Week 4:** Volunteer or event recap (Social)

### Pillar 4: Lifestyle / Brand
- **Weekly:** 1-2 behind-the-scenes or day-in-the-life posts
- **Monthly:** Longer personal reflection piece (Blog or Podcast)

## Batch Production Schedule

Content is produced in weekly batches, not day-of:

| Batch Day    | Activity                                             |
|-------------|------------------------------------------------------|
| Sunday PM   | Review next week's calendar, confirm topics           |
| Monday AM   | Film YouTube content (1-2 videos)                     |
| Monday PM   | Record podcast episode                                |
| Tuesday AM  | Write blog draft + email newsletter draft             |
| Tuesday PM  | Edit YouTube footage, create thumbnails               |
| Wednesday   | Edit podcast audio, write social copy, schedule posts |
| Thursday    | Review + approve all scheduled content                |
| Friday      | Engagement block, respond to comments, DMs, community |

---

## Quarterly Planning

At the start of each quarter, define:

1. **Quarter Theme** — One overarching narrative thread (e.g., "Q3: Building Your First Portfolio")
2. **Tentpole Content** — 2-3 high-production pieces that anchor the quarter
3. **Launch Windows** — Any product/service/event launches that need content support
4. **Collaboration Targets** — Guest episodes, cross-promotions, joint ventures
5. **Performance Review** — Pull top performers from last quarter, double down on what worked

## Calendar Database Properties

| Property         | Type         | Description                                      |
|------------------|-------------|--------------------------------------------------|
| Content Piece    | Relation     | Links to Content Pipeline DB                     |
| Publish Date     | Date         | Scheduled publish date and time                  |
| Platform         | Select       | Primary publish platform                         |
| Slot             | Select       | AM / PM / Evening                                |
| Status           | Select       | Planned / Confirmed / Published / Rescheduled    |
| Week Number      | Formula      | Auto-calculated from Publish Date                |
