# AI Prompt Library — Lumena OS

## Overview

Ready-to-use prompt templates for content creation across the Juniper Rose brand. Each prompt includes the brand voice prefix and is designed for use with OpenAI, Claude, or any LLM.

---

## Brand Voice Prefix (prepend to all prompts)

```
You are writing as Juniper Rose, a brand built by Alan Augustin focused on
real estate, wealth building, nonprofit work, and legacy media. The tone is
authoritative but warm, story-driven, direct, and always connects financial
topics back to family and community impact. Write in short paragraphs.
Avoid corporate jargon. Use "we" for brand content and "you" for educational
content. Every piece should have a clear hook and a call-to-action.
```

## 1. Video Hook Generator

**Use:** Generate opening hooks for YouTube videos and Reels.

```
Write 5 different opening hooks for a video about: [TOPIC]

Target audience: [AUDIENCE SEGMENT]
Video length: [SHORT (60s) / MEDIUM (5-10 min) / LONG (15+ min)]

Requirements:
- Each hook should be under 15 words (spoken in under 8 seconds)
- Use one of these patterns per hook:
  1. Contrarian statement ("Most people think X, but actually...")
  2. Curiosity gap ("There's one thing about X that nobody talks about...")
  3. Direct challenge ("If you're doing X, you need to stop right now")
  4. Story open ("Last week I lost $X because I didn't know this...")
  5. Stat shock ("X% of people will never Y — here's why")
- Make each hook specific, not generic
```

## 2. YouTube Script Outline

**Use:** Generate a structured script outline for long-form YouTube content.

```
Create a YouTube video script outline for: [TOPIC]

Target audience: [AUDIENCE SEGMENT]
Target length: [X] minutes
Primary keyword: [SEO KEYWORD]

Structure:
1. HOOK (0:00–0:15): Attention-grabbing opening
2. CONTEXT (0:15–1:00): Why this matters right now
3. MAIN CONTENT (1:00–[X-2]:00): 3–5 key points with examples
4. CTA (last 30s): What the viewer should do next

For each key point include:
- The point stated clearly in one sentence
- A real-world example or story
- A practical takeaway the viewer can use today

End with a CTA that drives to: [SUBSCRIBE / LINK / COMMENT / NEXT VIDEO]
```

## 3. Podcast Episode Outline

**Use:** Structure a podcast episode with segments and talking points.

```
Create a podcast episode outline for: [TOPIC]

Episode type: [SOLO / INTERVIEW / CO-HOST]
Target length: [X] minutes
Key message: [ONE SENTENCE summary of what the listener should walk away with]

Structure:
1. COLD OPEN (30s): Teaser clip from the best moment
2. INTRO (1 min): Episode topic + why it matters
3. SEGMENT 1: [Background/Context]
4. SEGMENT 2: [Deep dive / Main discussion]
5. SEGMENT 3: [Practical application / Action steps]
6. OUTRO: Key takeaway + CTA (rate/review/share)

For each segment, provide:
- 3–4 talking points
- 1 personal story or example to share
- 1 transition line to the next segment
```

## 4. Blog Post / SEO Article

**Use:** Generate a full blog post optimized for search and readability.

```
Write a blog post about: [TOPIC]

Primary keyword: [KEYWORD]
Secondary keywords: [KEYWORD 2], [KEYWORD 3]
Target word count: [1,200–2,000 words]
Target audience: [AUDIENCE SEGMENT]

Structure:
- Title: Include primary keyword, under 60 characters
- Meta description: 150–160 characters, include keyword
- H1: Title
- Introduction: Hook + problem statement + what they'll learn (150 words)
- H2 sections: 3–5 main points, each with:
  - Clear subheading (use keyword variations)
  - 200–300 words
  - At least one example, stat, or story
  - Bold key sentences for scannability
- Conclusion: Summary + single CTA
- Internal links: Suggest 2–3 places to link to other Juniper Rose content

Write for a reading level of grade 8. Use short paragraphs (2–3 sentences).
Include a bulleted list or numbered list in at least 2 sections.
```

## 5. Social Media Caption Generator

**Use:** Generate platform-specific captions.

```
Write a [PLATFORM] caption for: [TOPIC or CONTENT PIECE]

Platform: [Instagram / LinkedIn / X / TikTok]
Content type: [Reel / Carousel / Static post / Thread / Story]
Goal: [Engagement / Traffic / Awareness / Conversion]

Platform-specific rules:
- Instagram: Hook in first line (before "more"), emoji-light, 15–20 hashtags at end
- LinkedIn: Professional hook, 3–5 short paragraphs, end with question
- X: Punchy, under 280 chars for single tweet; 5–8 tweets for thread
- TikTok: Short, casual, 3–5 hashtags, trend-aware

Include a clear CTA: [COMMENT / SHARE / LINK IN BIO / DM ME / SAVE THIS]
```

## 6. Email Newsletter

**Use:** Draft a weekly newsletter send.

```
Write an email newsletter about: [MAIN TOPIC]

Subject line options: Provide 3 (under 50 characters each)
Preview text: 1 line that complements (not repeats) the subject

Structure:
- Opening: 1–2 sentences, personal and direct
- Big idea: The one thing the reader should know this week (3–4 paragraphs)
- Quick wins: 2–3 bullet points of actionable tips or resources
- CTA: One clear action (watch, read, reply, click)
- Sign-off: Brief, warm, personal

Tone: Like texting a smart friend who wants to level up.
Total length: 300–500 words.
```

## 7. Content Repurpose Prompt

**Use:** Transform one piece of content into another format.

```
Transform this [SOURCE FORMAT] into a [TARGET FORMAT]:

Source content:
[PASTE ORIGINAL CONTENT]

Target platform: [PLATFORM]
Target length: [WORD COUNT or DURATION]

Rules:
- Maintain the core message and key points
- Adapt the tone for [PLATFORM] (see brand voice guide)
- Add a hook appropriate for the target format
- Include a CTA appropriate for the target platform
- Do not copy sentences verbatim — rewrite for the new format
```

## 8. CTA Generator

**Use:** Generate calls-to-action for any content piece.

```
Generate 5 CTAs for: [CONTENT TOPIC]

Platform: [PLATFORM]
Goal: [SUBSCRIBE / CLICK LINK / COMMENT / SHARE / DM / BUY / SIGN UP]
Placement: [END OF VIDEO / CAPTION / EMAIL / BLOG / MID-CONTENT]

Requirements:
- Each CTA should be 1–2 sentences max
- Use action verbs (grab, drop, hit, share, save)
- Create urgency without being pushy
- Be specific about what they get (not "click here" but "grab the free checklist")
- Match the brand voice: direct, warm, no corporate speak
```

## 9. SEO Keyword Brief

**Use:** Research and plan SEO targeting for a content piece.

```
Create an SEO brief for: [TOPIC]

Include:
1. Primary keyword recommendation (with estimated search volume if known)
2. 5–8 secondary/related keywords
3. Search intent analysis (informational / transactional / navigational)
4. Suggested title tag (under 60 chars, keyword-first)
5. Meta description (150–160 chars)
6. H2 subheading suggestions (5–7) using keyword variations
7. FAQ section suggestions (3–5 questions people ask about this topic)
8. Internal linking opportunities to existing Juniper Rose content
9. Competing content analysis: What are the top 3 results doing well?
```

## 11. Daily Operations Brief

**Use:** Auto-generated morning brief (runs at 7:30 AM via automation).

```
Generate a daily operations brief for [DATE].

Inputs (pulled from Notion/CRM/Calendar):
- Today's calendar events: [LIST]
- Open tasks due today: [LIST]
- Hot leads requiring follow-up: [LIST]
- Content publishing today: [LIST]
- Yesterday's key metrics: [STATS]
- Pending approvals: [LIST]

Format:
🔥 TOP PRIORITY: [Most important thing today]

📅 SCHEDULE: [Chronological list of today's events]

📊 NUMBERS: [Key metrics from yesterday]

📝 TASKS: [Action items due today]

📢 CONTENT: [What's publishing / what needs review]

⚠️ ALERTS: [Anything overdue or flagged]
```

## Usage Notes

- Always prepend the **Brand Voice Prefix** to maintain consistent tone
- Replace all `[BRACKETED]` placeholders with actual values before running
- For best results, provide as much context as possible in the placeholders
- Save high-performing outputs as templates for future similar content
- Review all AI-generated content against the brand guardrails in `voice-config.md`
